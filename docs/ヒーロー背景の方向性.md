# ヒーロー背景「爽快」の方向性メモ

## いま実装したもの

- **グラデーション**: 空色（sky-50）→ 白 → 薄い緑（emerald-50）の斜めグラデ。明るく開けた印象。
- **大きなぼかし円**: 青・緑・アンバー系の円を3つ配置。単色ベタより奥行きと「風通しの良さ」を出す。
- **上部の光**: 上から少し青い光が差すような radial-gradient。
- **下の波型**: 白い波型の SVG でセクションを区切り、下のコンテンツへ自然につなぐ。

---

## まだイメージが固まらない場合に議論できるポイント

1. **色の方向**
   - もっと**青空寄り**（sky 強め）にするか
   - **ミント・緑**を強めて清涼感を出すか
   - **白ベース**で、アクセントだけ爽やか色にするか

2. **要素の量**
   - 今より**シンプル**（グラデ＋光だけ）にするか
   - **もっと動き**（細かい模様・幾何学）を足すか

3. **写真・イラスト**
   - キャンパス、空、窓の外など**写真を薄く背景に**するか
   - **イラスト**（人物・アイコン）を足して「就活サポート」感を出すか

4. **キーワードの具体化**
   - 「爽快」＝ 空・風・開けた窓・青空・清涼感・すっきり のどれに近いか
   - 参考にしたいサイトや画像があれば教えてもらえれば、それに寄せて調整できる

このメモをベースに「ここをもう少し〇〇に」と指定してもらえれば、文言やスタイルを細かく合わせられる。
